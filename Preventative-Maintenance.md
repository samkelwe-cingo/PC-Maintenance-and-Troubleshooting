# ✅ Preventative Maintenance Guide

## 📌 Overview

Preventative maintenance involves regular checks and actions performed to reduce system failures, improve performance, and extend the lifespan of computer equipment.

---

# 🗓️ Daily Maintenance Checks

☐ Confirm computer starts correctly  
☐ Check for unusual error messages  
☐ Verify internet connectivity  
☐ Confirm important applications are working  

---

# 📅 Weekly Maintenance Tasks

☐ Install pending Windows updates  
☐ Run security scans  
☐ Remove unnecessary temporary files  
☐ Review available storage space  
☐ Check for software updates  

---

# 📆 Monthly Maintenance Tasks

☐ Clean computer hardware externally  
☐ Check cables and connections  
☐ Review installed applications  
☐ Check backup status  
☐ Review system performance  

---

# 🔒 Security Best Practices

Recommended actions:

- Use strong passwords
- Keep software updated
- Avoid suspicious downloads
- Enable security protection
- Backup important files regularly

---

# 💻 Performance Optimization

Tasks:

☐ Manage startup programs  
☐ Remove unused software  
☐ Monitor system resources  
☐ Maintain sufficient storage space  
☐ Keep drivers updated
