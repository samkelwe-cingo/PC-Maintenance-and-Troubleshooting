# 🔧 Hardware Maintenance Guide

## 📌 Overview

This guide documents basic hardware maintenance procedures used to keep computer systems reliable and prevent common hardware failures.

---

# 🖥️ Internal Hardware Checks

## Dust Cleaning

Regular cleaning helps prevent overheating and improves system performance.

Steps:

☐ Shut down the computer completely  
☐ Disconnect power cables  
☐ Open the computer case safely  
☐ Remove dust using appropriate cleaning methods  
☐ Check cooling fans and airflow  

---

# 💾 Storage Maintenance

Check:

☐ Available disk space  
☐ Hard drive health  
☐ SSD performance  
☐ Unnecessary files  

Recommended actions:

- Remove temporary files
- Keep sufficient free storage space
- Monitor drive health

---

# 🧠 RAM Checks

Common RAM-related symptoms:

- Slow performance
- System freezing
- Application crashes
- Blue screen errors

Troubleshooting:

☐ Check installed RAM amount  
☐ Test memory if problems occur  
☐ Reseat RAM modules when necessary  

---

# 🌡️ Cooling System

Monitor:

- CPU temperature
- Fan operation
- Airflow

Possible overheating signs:

- Random shutdowns
- Loud fans
- Reduced performance

---

☐ Keyboard

☐ Monitor cables

☐ Printers and external devices
